1. Интерфейс (Interface) - Некоторый контракт, который обязан выполнить класс, который реализует данный интерфейс.
2. Содержит: Сигнатуры (свойств, методов или событий)

public int a;
public int A {get; set;}

public int doing(string a);

3. Нельзя!: 
- Прописывать реализацию членов интерфейса
- Создавать объект (но можно создать ссылку на интерфейс) "поговорим позже"
- создать конструктор
- не может содержать поля !(public int a;)
- не может сожержать перегрузку операторов
- объявлять методы как vitual или static, а так же менять модификатор доступа при их реализации в классе

4. По умолчанию все методы public
